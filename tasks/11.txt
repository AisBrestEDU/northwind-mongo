Create a SQL query to get Product list (name, unit price) where products cost between $5 and $15:

| ProductName | UnitPrice |
______________

db.products.aggregate([
    {
        $match: {
            "UnitPrice": { $gte: 5, $lte: 15 }
        }
    },
    {
        $project: {
            _id: 0,
            "ProductID": 1,
            "ProductName": 1
        }
    }
])
